# phae-compiler

### For Loop Guide  

#### Syntax  

```  
# Set the number of iterations  
x = 5; 

for i in range(x){ 
    expression;
}  
```  

#### Example  

```  
# Print numbers from 0 to 4  
x = 5;

for i in range(x){
    print(i);
}  
```  

#### Output  

```  
0  
1  
2  
3  
4  
```
